Cloud Entry: Firebase ဖြင့် Login စနစ် တည်ဆောက်ခြင်း 🔥

🎯 ရည်ရွယ်ချက်

Backend Code တွေအများကြီး ရေးစရာမလိုဘဲ စိတ်ချရတဲ့ User Login/Register စနစ်ကို ထည့်သွင်းတတ်ရန်။

✅ ရရှိမည့်အကျိုး

ကျောင်းသားတစ်ယောက်ချင်းစီမှာ ကိုယ်ပိုင်အကောင့်ရှိလာပြီး သူတို့ရဲ့ အမှတ်တွေကို Cloud ပေါ်မှာ သိမ်းဆည်းနိုင်တော့မယ်။

၁။ Firebase ဆိုတာ ဘာလဲ?

Firebase ဆိုတာ Google က ထုတ်ထားတဲ့ BaaS (Backend as a Service) တစ်ခုပါ။ ကျွန်တော်တို့က Frontend (React) ကိုပဲ အာရုံစိုက်ပြီး Database နဲ့ Login စနစ်တွေကို Firebase က အကုန်လုပ်ပေးပါတယ်။

၂။ Authentication (လူသိအောင်လုပ်ခြင်း)

Email နဲ့ Password သုံးပြီး Login ဝင်ဖို့အတွက် `firebase/auth` ကို သုံးရပါတယ်။

createUserWithEmailAndPassword(auth, email, password);
signInWithEmailAndPassword(auth, email, password);
