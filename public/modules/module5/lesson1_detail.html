Context API: Global မှတ်ဉာဏ် 🧠

🏠 သာဓက: ရွာထဲက အသံချဲ့စက်

စကားတစ်ခွန်းကို တစ်ယောက်ချင်းစီ လိုက်ပြောနေရတာ ပင်ပန်းပါတယ်။ (ဒါကို Prop Drilling လို့ ခေါ်ပါတယ်)။

Context API ဆိုတာကတော့ ရွာလယ်က အသံချဲ့စက်နဲ့ ကြေညာလိုက်သလိုပါပဲ။ ရွာသားအားလုံး (Components အားလုံး) ဟာ အိမ်ထဲကနေတင် အဲ့ဒီအသံကို တိုက်ရိုက် ကြားနိုင်ပါတယ်။ လိုချင်တဲ့ အချက်အလက်ကို အဆင့်ဆင့် တောင်းစရာမလိုဘဲ လေလှိုင်းထဲကနေ လှမ်းဖမ်းယူလိုက်ရုံပါပဲ။

"Login ဝင်ထားတဲ့ ကျောင်းသားနာမည်ကို Page တိုင်းမှာ ပြချင်ရင် Context API က အကောင်းဆုံး လက်နက်ပါပဲ။"
