🛠 Module 4: Data Doctor Arena

"useEffect Lifecycle နဲ့ API Fetching အမှားတွေကို ရှာဖွေပြီး ဒေတာတွေကို စနစ်တကျ ကိုင်တွယ်တတ်အောင် လေ့ကျင့်ပါ။"

Challenge 1: The Infinite Loop ♾️

"ကျောင်းသားက data ယူပြီး state update လုပ်လိုက်တာနဲ့ Browser က ဟန်း (Hang) သွားပါတယ်။ ဘာကြောင့်လဲ?"

BUGGY CODE useEffect(() => {
  fetchData(); // This updates state inside
}); // ❌ Missing dependency array
အဖြေမှန်ကို ကြည့်မည်
useEffect မှာ dependency array (ဒုတိယ parameter) မပါရင် component render ဖြစ်တိုင်း အလုပ်လုပ်ပါတယ်။ State update ဖြစ်ရင် render ပြန်ဖြစ်တဲ့အတွက် render -> useEffect -> state update -> render ဆိုပြီး Infinite Loop ပတ်သွားတာပါ။
✅ ပြင်ဆင်ရန်: useEffect(() => { ... }, []); // Empty array ထည့်ရပါမယ်။

Challenge 2: Async useEffect ⏳

"ကျောင်းသားက fetch လုပ်ချင်လို့ async/await သုံးလိုက်ပေမယ့် React က Warning Error တွေ ပြနေပါတယ်။"

BUGGY CODE useEffect(async () => { // ❌
  const data = await getData();
}, []);
အဖြေမှန်ကို ကြည့်မည်
useEffect ရဲ့ callback function ကို async တိုက်ရိုက်ပေးလို့ မရပါဘူး။ ဘာလို့လဲဆိုတော့ useEffect ဟာ Promise တစ်ခုကို return မပြန်ရဘဲ cleanup function (သို့မဟုတ်) ဘာမှမပြန်ရလို့ပါ။
✅ ပြင်ဆင်ရန်: useEffect ထဲမှာ async function တစ်ခု သီးသန့်ဆောက်ပြီး ပြန်ခေါ်ရပါမယ်။

Challenge 3: The Timer Leak 🛑

"ကျောင်းသားက စာမျက်နှာကို ပိတ်လိုက်ပေမယ့် နောက်ကွယ်မှာ Timer က ဆက်ပတ်နေပါတယ်။ Memory Leak ဖြစ်နေတာ ဘာကြောင့်လဲ?"

BUGGY CODE useEffect(() => {
  const timer = setInterval(() => console.log('Hi'), 1000);
  // ❌ Missing something here?
}, []);
အဖြေမှန်ကို ကြည့်မည်
Component တစ်ခုကို ဖျက်လိုက်တဲ့အခါ (Unmount) နောက်ကွယ်က timer တွေ၊ subscription တွေကို ပြန်ပိတ်ပေးဖို့ Cleanup Function လိုအပ်ပါတယ်။
✅ ပြင်ဆင်ရန်: return () => clearInterval(timer); လို့ return ပြန်ပေးရပါမယ်။

Challenge 4: The JSON Promise Error 📦

"ကျောင်းသားက data ကို ယူလိုက်ပေမယ့် Screen ပေါ်မှာ [object Promise] လို့ပဲ ပေါ်နေပါတယ်။"

BUGGY CODE const response = fetch(url);
const data = response.json(); // ❌
setData(data);
အဖြေမှန်ကို ကြည့်မည်
fetch ရော response.json() ရောဟာ အချိန်ယူရတဲ့ အလုပ်တွေဖြစ်လို့ Promise ကို return ပြန်ပါတယ်။ ဒါကြောင့် await သုံးဖို့ မေ့ခဲ့တာပါ။
✅ ပြင်ဆင်ရန်: const data = await response.json(); လို့ ရေးရပါမယ်။

Challenge 5: Stale Dependency 🔄

"Search term ပြောင်းသွားပေမယ့် API က data အသစ်ကို သွားမယူပါဘူး။ ဘာကြောင့်လဲ?"

BUGGY CODE useEffect(() => {
  fetchData(searchQuery);
}, []); // ❌ Something is missing in the array
အဖြေမှန်ကို ကြည့်မည်
useEffect ထဲမှာ အပြင်က state တစ်ခုခု (searchQuery) ကို သုံးထားရင် အဲ့ဒီ state ပြောင်းတိုင်း effect ပြန်ပတ်အောင် Dependency Array ထဲမှာ ထည့်ပေးရပါတယ်။
✅ ပြင်ဆင်ရန်: useEffect(() => { ... }, [searchQuery]);

Data Error (၅) ခုလုံးကို ပြင်နိုင်ပြီလား?

"API ချိတ်ဆက်မှုကို ကျွမ်းကျင်သွားပြီဆိုရင် မင်းရဲ့ App က တစ်ကမ္ဘာလုံးနဲ့ ချိတ်ဆက်နိုင်ပါပြီ။"
