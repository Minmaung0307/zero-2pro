၇.၂ - Performance Optimization: မြန်ဆန်သော App ဖြစ်လာစေရန်

၁။ Web Vitals ဆိုတာ ဘာလဲ?

Google က Website တစ်ခုရဲ့ အရည်အသွေးကို တိုင်းတာဖို့ Web Vitals လို့ခေါ်တဲ့ စနစ်ကို သုံးပါတယ်။ အဓိကအားဖြင့် Website က ဘယ်လောက်မြန်မြန် ပွင့်သလဲ၊ ခလုတ်နှိပ်ရင် ဘယ်လောက်မြန်မြန် တုံ့ပြန်သလဲဆိုတာတွေပဲ ဖြစ်ပါတယ်။ React App တွေမှာ Component တွေ အရမ်းများလာရင် App က လေးလာတတ်ပါတယ်။

၂။ React တွင် အကောင်းဆုံးဖြစ်အောင် ပြုပြင်ခြင်း

Professional တွေဟာ မလိုအပ်ဘဲ UI ပြန်ဆွဲတာ (Unnecessary Re-renders) ကို ရှောင်ကြဉ်ကြပါတယ်။ memo, useMemo, useCallback စတဲ့ Hooks တွေကို သုံးပြီး ကုဒ်တွေကို ပိုမိုမြန်ဆန်အောင် လုပ်ဆောင်ကြပါတယ်။ ပြီးတော့ ပုံတွေ (Images) ကို သုံးတဲ့အခါမှာလည်း အရွယ်အစား အကြီးကြီးတွေ မသုံးဘဲ WebP format လိုမျိုး ပေါ့ပါးတဲ့ ပုံစံတွေကို သုံးရပါမယ်။

Pro Tip:

Browser ရဲ့ Inspect Element ထဲက Lighthouse tab ကို သုံးပြီး သင့် Website ရဲ့ Performance Score ကို တိုင်းတာကြည့်ပါ။ ရမှတ် ၉၀ ကျော်ရင် Professional အဆင့်လို့ သတ်မှတ်နိုင်ပါတယ်။

ဤသင်ခန်းစာအတွက် ကိုယ်ပိုင်မှတ်စု
