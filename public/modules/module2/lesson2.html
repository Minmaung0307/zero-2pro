User ရိုက်သမျှကို လက်ခံခြင်း (Inputs) ⌨️

Website တစ်ခုမှာ User က စာရိုက်တာ၊ ခလုတ်နှိပ်တာတွေကို React က ဘယ်လို သိသလဲ? အဲ့ဒါကို Event Handling လို့ ခေါ်ပါတယ်။

onClick နှင့် onChange:

  • onClick: ခလုတ်နှိပ်တဲ့အခါ အလုပ်လုပ်စေချင်ရင် သုံးပါတယ်။
    <button onClick={() => alert('Hello')}>Click Me</button>
  • onChange: Input box ထဲမှာ စာရိုက်တဲ့အခါ အလုပ်လုပ်စေချင်ရင် သုံးပါတယ်။
    <input onChange={(e) => setName(e.target.value)} />

Pro Tip: Controlled Components

React မှာ Input ထဲကစာကို state ထဲ သိမ်းလိုက်တာကို Controlled Component လို့ ခေါ်ပါတယ်။ ဒါက အကောင်းဆုံး နည်းလမ်းပါပဲ။
