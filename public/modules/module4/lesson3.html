Loading & Error: ကျွမ်းကျင်သူတို့ရဲ့ အသေးစိတ် 🛠️

တကယ့် Professional Developer တစ်ယောက်ဟာ Data ကို ဒီအတိုင်း မပြပါဘူး။ Network နှေးနေရင် Loading Spinner ပြမယ်၊ Internet ပြတ်နေရင် Error Message ပြတတ်ရပါမယ်။

State ၃ ခု စီမံခြင်း:

  • Loading State: Data လာနေတုန်းမှာ User ကို စောင့်ခိုင်းတာ။
  • Error State: တစ်ခုခုမှားသွားရင် (ဥပမာ- 404) User ကို အကြောင်းကြားတာ။
  • Success State: Data တွေ အောင်မြင်စွာ ရောက်လာတဲ့အခါမှာပဲ UI ကို ပြတာ။
