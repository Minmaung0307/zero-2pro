Fetching Data: ပြင်ပကမ္ဘာနှင့် စကားပြောခြင်း 🌐

သင့်ရဲ့ App ထဲမှာ မရှိတဲ့ Data တွေ (ဥပမာ- သတင်းများ၊ စာရင်းများ) ကို အခြား Server တစ်ခုဆီကနေ သွားတောင်းတာကို API Fetching လို့ ခေါ်ပါတယ်။

Fetch API အသုံးပြုပုံ:

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data));

Pro Concept: Async/Await

ခေတ်မီ Developer တွေကတော့ `.then()` ထက် ပိုပြီး ဖတ်ရလွယ်တဲ့ `async/await` ကို ပိုသုံးကြပါတယ်။

🏋️ လက်တွေ့လေ့ကျင့်ခန်း:

JSONPlaceholder လိုမျိုး Free API တစ်ခုကနေ စာသားအချို့ကို ဆွဲထုတ်ပြီး သင့်ရဲ့ Screen ပေါ်မှာ ပေါ်လာအောင် လုပ်ဆောင်ကြည့်ပါ။ ဒါဟာ သင်ခန်းစာတွေကို Database ကနေ ဆွဲထုတ်ပြမယ့် ပုံစံအတိုင်းပါပဲ။

၄.၂ - API Fetching: ပြင်ပကမ္ဘာနှင့် စကားပြောခြင်း

၁။ Client နှင့် Server ဆက်သွယ်ပုံ

တကယ့် Web App တွေမှာ ဒေတာတွေ (ဥပမာ- သင်ခန်းစာစာရင်း၊ ကျောင်းသားအမှတ်များ) ဟာ ကိုယ့်စက်ထဲမှာ ရှိမနေပါဘူး။ အဝေးက Server ကြီးတစ်ခုပေါ်မှာ ရှိနေတာပါ။ အဲ့ဒီ Server နဲ့ ဆက်သွယ်ဖို့ API (Application Programming Interface) ကို သုံးရပါတယ်။ အလွယ်ပြောရရင် API ဟာ စားသောက်ဆိုင်က စားပွဲထိုးနဲ့ တူပါတယ်။ သင် (User) မှာချင်တဲ့ ဟင်း (Data) ကို မီးဖိုချောင် (Server) ထဲ သွားယူပေးတာ ဖြစ်ပါတယ်။

၂။ async နှင့် await (စောင့်ဆိုင်းခြင်း အတတ်ပညာ)

အင်တာနက်ကနေ ဒေတာယူတာဟာ ချက်ချင်းမပြီးပါဘူး။ Network နှေးတာ၊ ဒေတာများတာတွေကြောင့် စက္ကန့်ပိုင်းလောက် စောင့်ရပါတယ်။ အဲ့ဒီလို စောင့်ရတဲ့ အလုပ်တွေကို JavaScript မှာ Asynchronous လို့ ခေါ်ပါတယ်။ async နဲ့ await keyword တွေကို သုံးခြင်းအားဖြင့် ရှုပ်ထွေးလှတဲ့ code တွေကို ဖတ်ရလွယ်ကူစေပြီး ဒေတာရလာမှ နောက်တစ်ဆင့်ကို သွားဖို့ စနစ်တကျ ခိုင်းစေနိုင်ပါတယ်။
