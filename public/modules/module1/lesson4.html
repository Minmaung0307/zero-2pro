🛠 Module 1: Bug Hunter Arena

"ကုဒ်ထဲက အမှားတွေကို ရှာဖွေပြီး React Pro တစ်ယောက်ဖြစ်အောင် လေ့ကျင့်ပါ။"

Challenge 1: Component Naming

"ကျောင်းသားက ဒီ Component ကို ရေးလိုက်ပေမယ့် Browser မှာ Error တက်နေပါတယ်။ ဘာမှားနေလဲ?"

BUGGY CODE function welcomeCard() {
  return <h1>Hello React</h1>;
}
အဖြေမှန်ကို ကြည့်မည်
React မှာ Component နာမည်ဟာ အမြဲတမ်း အက္ခရာအကြီး (Capital Letter) နဲ့ စရပါမယ်။
✅ ပြင်ဆင်ရန်: function WelcomeCard() { ... }

Challenge 2: Multiple Elements

"ဒီ Component မှာ စာသား နှစ်ကြောင်း ပြချင်တာပါ။ ဒါပေမယ့် Error တက်နေပါတယ်။"

BUGGY CODE return (
  <h1>Name: Mg Mg</h1>
  <p>Age: 20</p>
);
အဖြေမှန်ကို ကြည့်မည်
JSX မှာ return ပြန်တဲ့အခါ Element တစ်ခုတည်း (Single Parent) အောက်မှာပဲ ရှိရပါမယ်။
✅ ပြင်ဆင်ရန်: <div> သို့မဟုတ် Fragments <> </> နဲ့ ပတ်ပေးရပါမယ်။

Challenge 3: Attribute Name

"ကျောင်းသားက UI ကို လှအောင် class ပေးထားပေမယ့် Console မှာ Warning တွေ တက်နေပါတယ်။"

BUGGY CODE <div class="active-profile">
  Student Information
</div>
အဖြေမှန်ကို ကြည့်မည်
JSX ဟာ JavaScript ပေါ် အခြေခံထားတာဖြစ်လို့ JavaScript ရဲ့ reserved word ဖြစ်တဲ့ class အစား className ကို သုံးရပါမယ်။
✅ ပြင်ဆင်ရန်: <div className="active-profile">

Challenge 4: Props Data Type

"ကျောင်းသားက Component ထဲကို ကိန်းဂဏန်း (Number) ပို့ချင်တာပါ။ ဒါပေမယ့် Error တက်နေပါတယ်။"

BUGGY CODE <StudentCard age=20 />
အဖြေမှန်ကို ကြည့်မည်
JSX မှာ String မဟုတ်တဲ့ အခြား Data Type တွေ (Numbers, Booleans, Objects) ကို ပို့ချင်ရင် တွန့်ကွင်း { } ထဲမှာ ထည့်ပေးရပါမယ်။
✅ ပြင်ဆင်ရန်: <StudentCard age={20} />

Challenge 5: Self-closing Tags

"ကျောင်းသားက HTML အတိုင်း Image ထည့်လိုက်ပေမယ့် React က Error ပြနေပါတယ်။"

BUGGY CODE <img src="profile.jpg">
အဖြေမှန်ကို ကြည့်မည်
JSX မှာ Tag အားလုံးဟာ အမြဲတမ်း ပိတ်ရပါမယ်။ Image လိုမျိုး အပိတ် tag မရှိတဲ့ အရာတွေကို Self-closing လုပ်ပေးရပါမယ်။
✅ ပြင်ဆင်ရန်: <img src="profile.jpg" />
