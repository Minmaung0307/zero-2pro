၇.၁ - Professional Styling & Design Systems

၁။ Website တစ်ခုကို ဘာက Professional ဆန်စေတာလဲ?

Developer အများစုဟာ Function တွေ အလုပ်လုပ်ဖို့ပဲ အာရုံစိုက်တတ်ကြပေမယ့် တကယ့် Professional တစ်ယောက်ကတော့ User Experience (UX) ကို အဓိကထားပါတယ်။ အမြင်လှရုံတင်မကဘဲ စာလုံးအရွယ်အစား (Typography)၊ အရောင်အသွေး (Color Palette) နဲ့ အကွာအဝေး (Spacing) တွေဟာ တစ်သမတ်တည်း ရှိနေဖို့ လိုပါတယ်။ ဒါကို Design System လို့ ခေါ်ပါတယ်။

Tailwind CSS ကို သုံးတဲ့အခါ px-4 လို့ တစ်နေရာမှာ သုံးရင် အလားတူ နေရာမျိုးတွေမှာလည်း px-4 ပဲ ဖြစ်နေရပါမယ်။ Professional ဆန်တဲ့ App တစ်ခုမှာ အရောင် ၃ မျိုးထက် ပိုမသုံးသင့်ပါဘူး။ (Primary, Secondary, and Accent colors)။ အခု ကျွန်တော်တို့ Portal မှာ သုံးထားတဲ့ Indigo နဲ့ Space Blue တို့ဟာ User ရဲ့ မျက်စိကို သက်သာစေသလို ခေတ်မီတဲ့ ခံစားချက်ကိုလည်း ပေးစွမ်းပါတယ်။

၂။ Dark Mode နှင့် User Preference

အခုခေတ်မှာ Dark Mode ဆိုတာ Option တစ်ခု မဟုတ်တော့ဘဲ Standard တစ်ခု ဖြစ်လာပါပြီ။ Tailwind ရဲ့ dark: variant ကို သုံးပြီး User ရဲ့ System setting အပေါ် မူတည်ပြီး အရောင်တွေ အလိုအလျောက် ပြောင်းလဲသွားအောင် လုပ်ဆောင်နိုင်ပါတယ်။ ဒါဟာ Developer တစ်ယောက်ရဲ့ အသေးစိတ် ဂရုစိုက်မှုကို ပြသရာ ရောက်ပါတယ်။

🏋️ လက်တွေ့လေ့ကျင့်ခန်း:

သင့်ရဲ့ Student Portal ထဲမှာ အောက်ပါ Design ပြင်ဆင်မှုတွေကို လုပ်ဆောင်ပါ -

  • ခလုတ်အားလုံး (Buttons) ကို transition နှင့် hover:scale-105 ထည့်ပြီး အသက်သွင်းပါ။
  • Card များအားလုံးကို backdrop-blur အာနိသင်ထည့်ပြီး Glassmorphism ပုံစံ ပြောင်းပါ။
  • စာသားများအတွက် Line-height (leading-relaxed) ကို သုံးပြီး ဖတ်ရလွယ်အောင် ပြင်ပါ။
