Dynamic Routing: ID များဖြင့် အလုပ်လုပ်ခြင်း 🆔

Website တစ်ခုမှာ သင်ခန်းစာပေါင်း ၁၀၀၀ ရှိရင် စာမျက်နှာ ၁၀၀၀ ထိုင်ဆောက်မနေပါနဲ့။ Dynamic Route ကို သုံးပြီး Page တစ်ခုတည်းကိုပဲ ID အလိုက် ဒေတာတွေ ပြောင်းလဲပြသလို့ ရပါတယ်။

အသုံးပြုပုံ:

<Route path="/lesson/:id" element={<LessonDetail />} />

ဒီနေရာမှာ `:id` ဆိုတာ ကိန်းဂဏန်း (သို့မဟုတ်) စာသား တစ်ခုခု ဖြစ်နိုင်ပါတယ်။

useParams Hook:

URL ထဲမှာ ပါလာတဲ့ ID ကို Component ထဲမှာ ပြန်ယူသုံးချင်ရင် `useParams` ကို သုံးရပါတယ်။

const { id } = useParams();
return <h1>သင်ကြည့်နေသော သင်ခန်းစာ ID မှာ {id} ဖြစ်သည်။</h1>;

Exercise: Lesson Detail Page

သင့်ရဲ့ App မှာ သင်ခန်းစာတစ်ခုကို နှိပ်လိုက်ရင် အဲ့ဒီသင်ခန်းစာရဲ့ အသေးစိတ်စာမျက်နှာဆီ URL မှာ ID ပါပြီး ရောက်သွားအောင် လုပ်ဆောင်ကြည့်ပါ။
