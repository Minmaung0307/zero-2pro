၇.၃ - Production Build နှင့် ကမ္ဘာသို့ လွှင့်တင်ခြင်း

၁။ Development vs Production

ကျွန်တော်တို့ ကုဒ်ရေးနေတဲ့အချိန် (npm run dev) မှာ ရရှိတဲ့ ကုဒ်တွေဟာ ဖိုင်ဆိုဒ် အကြီးကြီးတွေ ဖြစ်ပါတယ်။ သူတို့ထဲမှာ Debug လုပ်ဖို့အတွက် လိုအပ်တဲ့ အပိုအချက်အလက်တွေ အများကြီး ပါနေလို့ပါ။ ဒါပေမယ့် တကယ့် User တွေ သုံးဖို့အတွက် လွှင့်တင်တော့မယ်ဆိုရင် အဲ့ဒီအပိုတွေကို အကုန်ဖြုတ်ပစ်ရပါမယ်။

npm run build လို့ ရိုက်လိုက်တဲ့အခါ Vite က သင့်ရဲ့ React ကုဒ်တွေကို အစွမ်းကုန် ကျစ်လျစ်အောင် လုပ်ပေးပါတယ်။ ဒီအခါ dist folder ဆိုတာ ထွက်လာပါလိမ့်မယ်။ အဲ့ဒီ folder ထဲက ကုဒ်တွေဟာ ကမ္ဘာပေါ်မှာ အမြန်ဆုံး အလုပ်လုပ်နိုင်တဲ့ ကုဒ်တွေပဲ ဖြစ်ပါတယ်။

၂။ Custom Domain နှင့် SSL

သင့်ရဲ့ Website address က .web.app နဲ့ ဆုံးနေတာထက် yourname.com လိုမျိုး ကိုယ်ပိုင် domain နဲ့ ဖြစ်နေတာက ပိုပြီး ယုံကြည်မှု (Credibility) တိုးစေပါတယ်။ Firebase Hosting မှာ သင့်ရဲ့ ကိုယ်ပိုင် Domain ကို အလွယ်တကူ ချိတ်ဆက်နိုင်သလို SSL (HTTPS) လုံခြုံရေးကိုလည်း အခမဲ့ ပေးထားပါတယ်။

🎓 သင်တန်းဆင်းခြင်း အထိမ်းအမှတ်

Module ၁ ကနေ ၇ အထိ အောင်မြင်စွာ တက်လှမ်းနိုင်ခဲ့တဲ့အတွက် ဂုဏ်ယူပါတယ်။ အခုဆိုရင် သင်ဟာ အခြေခံကနေ လက်တွေ့အလုပ်ခွင်ဝင်နိုင်တဲ့အထိ အရည်အချင်းရှိတဲ့ React Developer တစ်ယောက် ဖြစ်သွားပါပြီ။
