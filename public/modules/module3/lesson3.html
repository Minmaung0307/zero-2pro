Redirecting & Security 🛡️

useNavigate:

Link ကို မနှိပ်ဘဲ Code အနေနဲ့ Page တစ်ခုကို သွားခိုင်းတာပါ။ ဥပမာ - Login အောင်မြင်သွားရင် Dashboard ကို အလိုအလျောက် ပို့ပေးချင်တဲ့အခါ သုံးပါတယ်။

navigate('/dashboard');

Protected Routes:

တချို့စာမျက်နှာတွေ (ဥပမာ- Grade စစ်တဲ့ Page) ကို Login ဝင်ထားတဲ့သူပဲ ကြည့်လို့ရအောင် ကာကွယ်ထားတဲ့ နည်းလမ်းဖြစ်ပါတယ်။ ဒါဟာ Real-world Web App တွေရဲ့ လုံခြုံရေးအတွက် အရေးကြီးဆုံး အပိုင်းပါ။
