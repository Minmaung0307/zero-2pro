Lesson 2: Detail - JSX နဲ့ Components ကို နားလည်ခြင်း 🧱

🏠 သာဓက: Lego တုံးလေးများ

အရင်က Website ရေးရင် စာရွက်အကြီးကြီးတစ်ရွက်ပေါ်မှာ ပန်းချီဆွဲသလိုမျိုး တစ်ခုတည်း အရှည်ကြီး ရေးရပါတယ်။

React ကတော့ Website ကို Lego (လီဂို) တုံးလေးတွေလို အပိုင်းပိုင်းခွဲလိုက်တာပါ။ ဥပမာ- "ခေါင်းစဉ်တုံး"၊ "ဘေးဘားတုံး"၊ "အောက်ခြေတုံး" ဆိုပြီး သီးသန့်စီ ဆောက်ပါတယ်။ ဒါကို Components လို့ ခေါ်ပါတယ်။

❓ JSX ဆိုတာ ဘာလဲ?

"JavaScript ထဲက HTML အတုအယောင်"

တကယ်တော့ Browser က JSX ကို နားမလည်ပါဘူး။ ဒါပေမယ့် React က ကျွန်တော်တို့ ရေးရလွယ်အောင် JavaScript ထဲမှာ HTML ပုံစံမျိုး ရေးခွင့်ပေးထားတာပါ။ ဒါကို Babel ဆိုတဲ့ စနစ်က နောက်ကွယ်မှာ Browser နားလည်တဲ့ ကုဒ်အဖြစ် ပြောင်းပေးပါတယ်။

ဘာကြောင့် အကျိုးရှိလဲ?

  • Reusability: တစ်ခါဆောက်ထားတဲ့ Button တုံးလေးကို Page တိုင်းမှာ ပြန်သုံးလို့ရတယ်။
  • Clean Code: ကုဒ်တွေက အပိုင်းလိုက်လေးတွေမို့ ဖတ်ရတာ ရှင်းလင်းတယ်။
