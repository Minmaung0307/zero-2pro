Module 1 • Lesson 1.3

Props: Component တွေဆီ ဒေတာပို့နည်း 📦

Component တစ်ခုကို ရေးထားပြီး ပုံစံတူ ဒေတာမတူအောင် ဖန်တီးချင်ရင် Props ကို သုံးရပါတယ်။ ဥပမာ - ကျောင်းသားကတ် (Student Card) ဆိုတဲ့ Component တစ်ခုကို ရေးထားမယ်။ အဲ့ဒီထဲကို နာမည်တွေ၊ အတန်းတွေကို လှမ်းပို့ပေးတာပါ။

ဥပမာ:

// Parent Component (App.jsx)
<StudentCard name="မောင်မောင်" class="Grade 10" />
<StudentCard name="မမောင်" class="Grade 11" />

// Child Component (StudentCard.jsx)
function StudentCard(props) {
  return <h1>နာမည်: {props.name}</h1>;
}

🎓 Module 1 Final Project

ဒီ Module မှာ သင်ယူခဲ့တဲ့ Component နဲ့ Props ကို သုံးပြီး Student Portfolio Portal တစ်ခုကို React နဲ့ တည်ဆောက်ကြည့်ပါ။ ကျောင်းသား ၃ ယောက်ရဲ့ Card လေးတွေ ပေါ်လာအောင် လုပ်ဆောင်ရမှာ ဖြစ်ပါတယ်။
