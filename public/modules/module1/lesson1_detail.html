Lesson 1: Detail - နက်နဲသော အခြေခံများကို နားလည်ခြင်း 🧠

သာဓက- အိမ်တစ်လုံး ဆောက်ကြစို့

React Web App တစ်ခု တည်ဆောက်တာဟာ အဆင့်မြင့် အိမ်တစ်လုံး ဆောက်တာနဲ့ တူပါတယ်။ ဘာကြောင့် ဒီ Tool တွေကို အရင်လိုတာလဲဆိုရင်-

၁။ Node.js (မြေနေရာနှင့် လျှပ်စစ်မီး)

Node.js ဟာ အိမ်ဆောက်မယ့် "မြေနေရာ" နဲ့ "လျှပ်စစ်မီး" လိုမျိုး အခြေခံ စနစ်ကြီးပါ။ သူ မရှိရင် ကျန်တဲ့ စက်ပစ္စည်းတွေ (Tools တွေ) ဘာမှ ခိုင်းလို့ မရပါဘူး။

၂။ NPM (ပစ္စည်းပို့ဆောင်ရေး ဝန်ဆောင်မှု)

NPM ဆိုတာ "အုတ်၊ သဲ၊ ကျောက်" လိုအပ်တဲ့ ပစ္စည်းတွေကို ဆိုင်ကနေ မှာယူပေးမယ့် ပစ္စည်းပို့သမားပါ။ npm install ဆိုတာ ဈေးဝယ်စာရင်းအတိုင်း ပစ္စည်းတွေ အိမ်ထဲ ရောက်အောင် မှာယူလိုက်တာ ဖြစ်ပါတယ်။

၃။ Vite (အမြန်ဆုံး အိမ်ဆောက်စက်ယန္တရား)

အိမ်တစ်လုံးကို လက်နဲ့ တစ်တုံးချင်း စီမနေတော့ဘဲ ခေတ်မီစက်ကြီးတွေနဲ့ အမြန်ဆောက်တာကို Vite လို့ ခေါ်ပါတယ်။ သူက အိမ်ရဲ့ "ပုံစံခွက်" ကို ချက်ချင်း ထုတ်ပေးပါတယ်။

၄။ React (အတွင်းပိုင်း ဒီဇိုင်းနှင့် အလှဆင်ခြင်း)

အိမ်ကြီး ဖြစ်သွားပြီဆိုရင် အထဲမှာ မီးဖိုချောင်၊ ဧည့်ခန်း ဆိုပြီး "အစိတ်အပိုင်း (Components)" လေးတွေနဲ့ အလှဆင်တာက React ရဲ့ အလုပ်ပါ။

Command များ၏ အသေးစိတ် ရှင်းလင်းချက်

Q1: "create vite@latest" ဆိုတာ Vite ကို ထည့်သွင်းတာလား?

// မဟုတ်ပါဘူး။

ဒါဟာ Vite ဆိုတဲ့ "အိမ်ပုံစံထုတ်တဲ့ စက်" ကို ခဏငှားသုံးပြီး Project ပုံစံခွက် (Folder Structure) ကို ဆောက်ခိုင်းလိုက်တာပါ။ သင့်စက်ထဲမှာ Vite ကို အပြီးအပိုင် Install လုပ်လိုက်တာ မဟုတ်ဘဲ၊ လိုအပ်တဲ့ ဖိုင်တွဲတွေကို ဖန်တီးခိုင်းလိုက်တာ ဖြစ်ပါတယ်။

Q2: "--template react" ဆိုတာ React ကို ထည့်သွင်းတာလား?

// ဒါက ရွေးချယ်မှု (Choice) တစ်ခုပါ။

Vite ဟာ Vue, Svelte စတဲ့ တခြားပုံစံတွေလည်း ဆောက်ပေးနိုင်ပါတယ်။ "--template react" လို့ ပြောလိုက်တာဟာ "ငါဆောက်မယ့် အိမ်ပုံစံကို React နည်းပညာနဲ့ပဲ ဆောက်ပေးပါ" လို့ လမ်းညွှန်ချက် ပေးလိုက်တာပါ။

Q3: "npm install" ဆိုတာ Node ကို ထည့်သွင်းလိုက်တာလား?

// လုံးဝ မဟုတ်ပါဘူး။ အရေးကြီးပါတယ်။

"npm install" ကို ရိုက်ဖို့အတွက် သင့်စက်ထဲမှာ Node.js ရှိနှင့်ပြီးသား ဖြစ်ရပါမယ်။

သူ့ရဲ့ အလုပ်က `package.json` (ဈေးဝယ်စာရင်း) ထဲမှာ ပါတဲ့ React Library ဖိုင်တွေကို အင်တာနက်ပေါ်ကနေ ဆွဲချပြီး သင့်ရဲ့ `node_modules` folder ထဲကို ထည့်ပေးလိုက်တာ ဖြစ်ပါတယ်။

ဘာ့ကြောင့် အရင်ဆုံး ထည့်သွင်းရတာလဲ?

အုတ်၊ သဲ၊ ကျောက် မရှိဘဲ အိမ်ဆောက်လို့ မရသလိုပဲ၊ Node, Vite နဲ့ React မရှိဘဲနဲ့ ခေတ်မီ Web App တစ်ခုကို မတည်ဆောက်နိုင်လို့ပါ။ ဒါတွေဟာ Developer တစ်ယောက်အတွက် "လက်နက်ကိရိယာတွေ" ပါပဲ။ ဒါတွေ အဆင်သင့်ဖြစ်ပြီဆိုမှ ကျွန်တော်တို့ ကုဒ်စရေးလို့ ရမှာ ဖြစ်ပါတယ်။
