🛠 Module 3: Navigation Fixer Arena

"React Router ရဲ့ လမ်းကြောင်းအမှားတွေကို ရှာဖွေပြီး Single Page Application (SPA) တစ်ခုကို ကျွမ်းကျင်စွာ တည်ဆောက်ပါ။"

Challenge 1: The Page Reload Bug

"ကျောင်းသားက About Page ကို သွားဖို့ Link ချိတ်ထားပေမယ့် နှိပ်လိုက်တိုင်း Browser က Refresh ဖြစ်နေပါတယ်။ ဘာမှားနေလဲ?"

BUGGY CODE function Navbar() {
  return (
    <nav>
      <a href="/about">About Page</a> // ❌
    </nav>
  );
}
အဖြေမှန်ကို ကြည့်မည်
React Router မှာ <a href="..."> ကို သုံးရင် Browser က Server ဆီကနေ Page အသစ်ကို တောင်းဆိုတဲ့အတွက် Refresh ဖြစ်သွားပါတယ်။ SPA ပုံစံ Refresh မဖြစ်ချင်ရင် Link component ကို သုံးရပါမယ်။
✅ ပြင်ဆင်ရန်: <Link to="/about">About Page</Link>

Challenge 2: Relative vs Absolute Path

"ဒီ Route က အလုပ်မလုပ်ဘဲ Page အလွတ်ကြီးပဲ ပေါ်နေပါတယ်။ URL လမ်းကြောင်းမှာ ဘာလိုနေလဲ?"

BUGGY CODE <Route path="dashboard" element={<Dashboard />} /> // ❌
အဖြေမှန်ကို ကြည့်မည်
Main Route တွေ သတ်မှတ်တဲ့အခါ လမ်းကြောင်းရဲ့ ရှေ့မှာ Forward Slash (/) ထည့်ပေးဖို့ လိုအပ်ပါတယ်။ ဒါမှသာ root ကနေ စတင်တဲ့ လမ်းကြောင်းအဖြစ် React က သိမှာပါ။
✅ ပြင်ဆင်ရန်: path="/dashboard"

Challenge 3: useParams Usage

"URL ထဲက id ကို ယူသုံးချင်တာပါ။ ဒါပေမယ့် Screen ပေါ်မှာ [object Object] လို့ပဲ ပေါ်နေပါတယ်။"

BUGGY CODE const id = useParams(); // ❌
return <h1>Lesson ID: {id}</h1>;
အဖြေမှန်ကို ကြည့်မည်
useParams() hook ဟာ Object တစ်ခုကို Return ပြန်ပေးတာပါ။ အဲ့ဒီထဲကမှ ကျနော်တို့ လိုချင်တဲ့ variable နာမည်ကို Destructuring လုပ်ပြီး ယူရပါမယ်။
✅ ပြင်ဆင်ရန်: const { id } = useParams();

Challenge 4: The Router Wrapper

"ကျောင်းသားက Routes တွေကို ရေးလိုက်ပေမယ့် Browser မှာ 'useRoutes() may be used only in the context of a Router component' ဆိုတဲ့ Error ကြီး တက်နေပါတယ်။"

BUGGY CODE // App.js
<Routes>
  <Route path="/" element={<Home />} />
</Routes>
အဖြေမှန်ကို ကြည့်မည်
React Router ရဲ့ Component တွေ (Routes, Route, Link) အားလုံးဟာ BrowserRouter ရဲ့ အောက်မှာ (Inside) ရှိနေရပါမယ်။
✅ ပြင်ဆင်ရန်: App တစ်ခုလုံးကို <BrowserRouter> နဲ့ ပတ်ပေးရပါမယ်။

Challenge 5: Dynamic Route Definition

"သင်ခန်းစာ ID အလိုက် Page တွေကို ပြချင်တာပါ။ ဒါပေမယ့် ဘယ် ID ရိုက်ရိုက် Page မပေါ်လာပါဘူး။"

BUGGY CODE <Route path="/lesson/id" element={<Details />} /> // ❌
အဖြေမှန်ကို ကြည့်မည်
Dynamic Parameter သတ်မှတ်ချင်ရင် ရှေ့မှာ Colon (:) ထည့်ပေးရပါတယ်။ အဲ့ဒါမှသာ React က ဒါဟာ variable တစ်ခုဖြစ်မှန်း သိမှာပါ။
✅ ပြင်ဆင်ရန်: path="/lesson/:id"

Navigation Error (၅) ခုလုံးကို ပြင်နိုင်ပြီလား?

"Routing ကို နားလည်သွားပြီဆိုရင် မင်းရဲ့ App က စာမျက်နှာတစ်ခုတည်း မဟုတ်တော့ပါဘူး။"
