Optimization: ကုဒ်များကို အဆီချခြင်း 🏎️

🏠 သာဓက: စားသောက်ဆိုင် မီးဖိုချောင်၏ စွမ်းဆောင်ရည်

စားဖိုမှူးတစ်ယောက်ဟာ ဟင်းတစ်ပွဲချက်တိုင်း ဈေးကိုအပြေးသွားပြီး ပစ္စည်းအသစ် ပြန်ဝယ်နေရင် အဲ့ဒီဆိုင်ဟာ အရမ်းနှေးနေပါလိမ့်မယ်။ အမြဲသုံးတဲ့ ပစ္စည်းတွေကို ရေခဲသေတ္တာထဲမှာ အဆင်သင့် ထည့်ထားရပါမယ်။

React မှာလည်း Re-rendering ဆိုတာ အဲ့ဒီလိုပါပဲ။ Component တစ်ခုက ဒေတာမပြောင်းဘဲနဲ့ ခဏခဏ ပြန်ဆွဲနေရင် App က လေးလာပါမယ်။ useMemo နဲ့ useCallback ဆိုတာ အဲ့ဒီဟင်းသီးဟင်းရွက်တွေကို ရေခဲသေတ္တာထဲ ထည့်ထားသလိုပါပဲ။ မပြောင်းလဲသေးတဲ့ ရလဒ်တွေကို မှတ်ဉာဏ်ထဲမှာ သိမ်းထားပြီး လိုအပ်မှသာ ပြန်ဆွဲတာ ဖြစ်လို့ App က ပြိုင်ကားတစ်စီးလို မြန်ဆန်နေမှာ ဖြစ်ပါတယ်။

Lighthouse နှင့် အရည်အသွေးတိုင်းတာခြင်း

Professional တစ်ယောက်ဟာ "ငါ့ Website က မြန်တယ်" လို့ ခံစားချက်နဲ့ မပြောပါဘူး။ ကိန်းဂဏန်းနဲ့ သက်သေပြပါတယ်။ Browser ရဲ့ Inspect Tool ထဲက Lighthouse ဆိုတာ သင့် Website ရဲ့ ကျန်းမာရေးကို စစ်ဆေးပေးတဲ့ ဆရာဝန်ပါပဲ။ သူက SEO, Accessibility နဲ့ Performance တို့ကို အမှတ်ပေးပါတယ်။ အမှတ် ၉၀ ကျော်ဖို့ဆိုရင် ပုံတွေကို Compress လုပ်ရမယ်၊ မလိုတဲ့ Library တွေကို ဖယ်ရှားရမယ်ဆိုတဲ့ အဆင့်မြင့် ပညာရပ်တွေကို ဒီသင်ရိုးမှာ လက်တွေ့လေ့လာရမှာ ဖြစ်ပါတယ်။
